== Description ==

PushQuote turns your pullquotes into quotes your readers can share on Twitter and Facebook.

Enjoy !

== Installation ==

1. Copy the "realtidbits-pushquote" folder into your wordpress "wp-content/plugins/" folder using your favorite FTP program.
2. Activate the plugin through the 'Plugins' menu in WordPress

Once the plugin has been activated you will notice a new button in the post editor to add pullquotes. Also, you will need to set your bit.ly credentials in the WP general settings for the twitter and facebook share buttons.
